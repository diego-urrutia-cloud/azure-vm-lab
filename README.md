# Azure Virtual Machine Lab

This project demonstrates how to deploy a basic virtual machine in Microsoft Azure.

## Objective

Learn the fundamentals of cloud infrastructure.

## Services Used

- Azure Virtual Network
- Azure Subnet
- Azure Virtual Machine

## Architecture

Internet
   │
Virtual Network
   │
Subnet
   │
Virtual Machine

## Steps

1. Create a Virtual Network
2. Create a Subnet
3. Deploy a Virtual Machine
4. Connect to the VM
