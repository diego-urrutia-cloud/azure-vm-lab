# Deployment Steps

## Step 1 – Create Virtual Network

Go to the Azure Portal and create a Virtual Network.

Name: lab-vnet

## Step 2 – Create Subnet

Inside the virtual network create a subnet.

Name: lab-subnet

## Step 3 – Deploy Virtual Machine

Create a Linux virtual machine.

Name: lab-vm

## Step 4 – Connect to the VM

Use SSH or RDP to connect to the machine.
